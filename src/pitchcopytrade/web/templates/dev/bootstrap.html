local dev bootstrap

Один локальный доступ для staff и Mini App

Эта панель работает только в `development` / `test` / `local` и поднимает один bootstrap-аккаунт с ролями `admin`, `author`, `moderator` и Telegram-cookie для `/app/*`.

Admin
Откроет `/admin/dashboard` и оставит доступ к `app/*` включённым.
Author
Откроет `/author/dashboard` и сохранит доступ к `app/*`.
Moderator
Откроет `/moderation/queue` и сохранит доступ к `app/*`.
Catalog
Откроет `/app/catalog` с уже готовыми staff-cookie и Telegram-cookie.
