Краткая HTTPS-only инструкция для входа в админку, кабинет автора и пользовательский Telegram-flow.

1. Администратор

  1. Откройте https://pct.test.ptfin.ru/login.
  2. Введите логин admin1.
  3. Введите пароль admin-demo-pass.
  4. После входа откроется /admin/dashboard.
Что проверить
  • список стратегий;
  • список продуктов подписки;
  • очередь платежей;
  • подтверждение pending payment.

2. Автор стратегии

  1. Откройте https://pct.test.ptfin.ru/login.
  2. Введите логин author1.
  3. Введите пароль author-demo-pass.
  4. После входа откроется /author/dashboard.
Что проверить
  • список своих рекомендаций;
  • создание и редактирование рекомендации;
  • structured legs;
  • вложения и статусы идей.

3. Пользователь в Telegram

  1. Откройте бота @Avt09_Bot.
  2. Отправьте /start.
  3. Отправьте /catalog.
  4. Для создания заявки отправьте /confirm_buy momentum-ru-month.
  5. После подтверждения оплаты администратором отправьте /feed.
Ожидаемый результат
  • после /confirm_buy появляется заявка на оплату;
  • администратор подтверждает платеж в web-кабинете;
  • после этого /feed показывает доступные рекомендации.

4. Полный тестовый поток

  1. Пользователь в Telegram: /start → /catalog → /confirm_buy momentum-ru-month.
  2. Администратор: входит как admin1 и подтверждает новый pending payment.
  3. Автор: входит как author1 и проверяет кабинет автора.
  4. Пользователь в Telegram: отправляет /feed и получает рекомендацию.

Документ рассчитан только на HTTPS-стенд: https://pct.test.ptfin.ru.
